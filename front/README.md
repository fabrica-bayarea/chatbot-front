# Eda / Frontend

## Rodando localmente

Clone o projeto:

```bash
  git clone https://github.com/fabrica-bayarea/eda.git
```

Mude para a pasta recém-criada:

```bash
  cd eda/front
```

Instale as dependências:

```bash
  npm install
```

Inicie a aplicação:

```bash
  npm run dev
```
